# 🤖 ShopBot - AI Customer Service Chatbot

An intelligent customer service chatbot that understands user intent, extracts important information, remembers conversation context, and escalates frustrated customers to human agents.

## 🎯 Features

| Feature | Description |
|---------|-------------|
| **11+ Intents** | Greeting, order status, returns, payments, shipping, discounts, account, human agent, and more |
| **Entity Extraction** | Automatically detects order IDs (ORD-12345), emails, and phone numbers |
| **Context Tracking** | Remembers conversation across multiple turns |
| **Sentiment Detection** | Detects negative keywords and offers human agent escalation |
| **Dual Interface** | Python CLI tool + Beautiful web chat interface |
| **No Backend Required** | Web version runs entirely in the browser |
